# Student Performance Analytics & Risk Assessment

## Overview
This project analyzes student performance data to identify factors affecting academic outcomes and classify students into risk categories.

## Features
- Exploratory Data Analysis (EDA)
- Subject-wise Performance Analysis
- Correlation Heatmap
- Test Preparation Impact Analysis
- Risk Classification System

## Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Findings
- Reading and Writing scores show strong positive correlation.
- Students completing test preparation generally perform better.
- Students are classified into High, Medium, and Low Risk groups based on overall performance.
